# soulseek

### install with curl

```bash
curl raw.github.com/jeromedecoster/soulseek/master/install.sh \
    --location \
    --silent \
    | bash
```

### install with wget

```bash
wget raw.github.com/jeromedecoster/soulseek/master/install.sh \
    --output-document=- \
    --quiet \
    | bash
```
